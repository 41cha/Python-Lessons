# Module 06: Strings

У цьому модулі студенти вчаться працювати з **рядками** в Python: змінювати регістр, обрізати пробіли, шукати та замінювати підрядки, розбивати текст на слова, перевіряти паліндроми та робити просту валідацію email.

## Навчальні цілі

Після проходження модуля студент зможе:

- Використовувати базові методи рядків: `len`, `lower`, `upper`, `strip`, `replace`, `split`, `join`. 
- Перебирати символи в рядку та рахувати потрібні символи (голосні, частоти). 
- Перетворювати рядок у список слів і назад.
- Реалізувати прості текстові перевірки: паліндром, примітивна валідація email.

## Основні операції з рядками

### Довжина, регістр

- `len(text)` — повертає довжину рядка (кількість символів).
- `text.upper()` — повертає копію рядка у ВЕРХНЬОМУ регістрі. 
- `text.lower()` — повертає копію рядка в нижньому регістрі. 

Це корисно для аналізу тексту та регістр-незалежних перевірок (наприклад, підрахунок голосних). 

### Обрізання пробілів

- `text.strip()` — прибирає пробіли (та інші пробільні символи) з початку і кінця рядка. 

Зручно для обробки введення користувача, де часто бувають «зайві» пробіли.  

### Пошук і заміна

- `text.replace(old, new)` — повертає копію рядка, де всі входження `old` замінені на `new`. 

Наприклад, заміна `"Python"` на `"JavaScript"` у всьому реченні.  

### split і join

- `text.split()` — розбиває рядок на список слів (за пробілами за замовчуванням). 
- `"sep".join(list_of_strings)` — об’єднує список рядків в один рядок, вставляючи між ними роздільник `sep`.

Приклад:

```python
text = "Python is fun"
words = text.split()           # ["Python", "is", "fun"]
result = "-".join(words)       # "Python-is-fun"
```

Це типова операція для обробки тексту та формування форматованого виводу.

## Огляд вправ модуля

### Task 1: Basic string info

- Ввід: довільний рядок від користувача.  
- Вивід:
  - довжина рядка;  
  - той же текст у верхньому та нижньому регістрах.  
- Цілі: знайомство з `len`, `upper`, `lower` і базовою роботою з введенням.

### Task 2: Trim spaces

- Ввід: рядок з пробілами на початку та в кінці.  
- Вивід:
  - оригінальний рядок у лапках;  
  - обрізаний рядок у лапках.  
- Цілі: зрозуміти, як `strip()` очищує введення й чому важливо показувати різницю явно (через лапки).

### Task 3: Count vowels

- Ввід: рядок.  
- Поведінка: підрахувати кількість голосних літер (`a, e, i, o, u, y`), ігноруючи регістр.  
- Цілі:
  - перетворення до нижнього регістру для уніфікації;  
  - перебір символів у циклі;  
  - перевірка `if ch in "aeiouy"`. 

### Task 4: Replace word

- Текст: `"Python is hard, but Python is fun"`.  
- Поведінка: замінити всі `"Python"` на `"JavaScript"`.  
- Цілі: використання `replace()` для глобальної заміни підрядка в рядку. 

### Task 5: Split and join

- Текст: `"Python is fun and powerful"`.  
- Поведінка:
  - розбити рядок на слова;  
  - зібрати назад у вигляді `"Python-is-fun-and-powerful"`.  
- Цілі:
  - перетворення рядка в список слів (`split()`);  
  - формування нового рядка з іншим роздільником через `join()`. 

### Task 6: Simple email validator (basic)

- Ввід: рядок `email`.  
- Перевірка:
  - чи містить рядок `"@"`;  
  - чи є крапка `"."` **після** `"@"`.  
- Вивід: `"Valid email"` / `"Invalid email"`.  
- Цілі:
  - робота з індексами та методом `find()`;  
  - розуміння дуже базової валідації строкового формату (без складного regex). 

### Task 7: Palindrome checker

- Ввід: рядок (може містити пробіли, різні регістри).  
- Кроки:
  - прибрати пробіли;  
  - перевести до нижнього регістру;  
  - перевірити, чи дорівнює рядок своїй розвернутій версії (`[::-1]`).  
- Вивід: `"Palindrome"` / `"Not a palindrome"`.  
- Цілі:
  - очистка рядка перед перевіркою;  
  - використання індексування зі зрізом для розвороту;  
  - практика простої текстової логіки.

### Task 8 (optional): Character frequency

- Ввід: рядок.  
- Поведінка:
  - порахувати, скільки разів зустрічається кожен символ;  
  - вивести у форматі `"символ -> кількість"`.  
- Цілі:
  - комбінація рядків і словників;  
  - класичний патерн частотного словника (frequency dictionary) для тексту.
