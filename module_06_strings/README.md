# Module 06: Strings

Рядки в Python — це базовий спосіб працювати з текстом: імена користувачів, повідомлення, введення з клавіатури, JSON, дані з файлів чи мережі — майже все це спочатку приходить як **текст**. У цьому модулі важливо не просто вивчити методи рядків, а зрозуміти, як із їх допомогою будувати прості текстові трансформації.

## Що таке рядок у Python

Рядок — це **послідовність символів** у лапках:

```python
text = "Hello, world!"
```

Він може містити літери, цифри, пробіли, символи пунктуації та навіть емодзі.

Важливі властивості: 

- рядки **незмінні** (immutable): методи повертають новий рядок, а не змінюють існуючий;  
- до символів можна звертатися за індексом: `text[0]`;  
- рядок — це «звичайний» об’єкт Python з купою корисних методів (`lower`, `upper`, `strip`, `replace`, `split`, `join` та ін.).

***

## Основні операції над рядками

### Довжина і регістр

Коли ви працюєте з текстом, часто потрібно:

- дізнатися довжину: `len(text)`;  
- зробити текст **незалежним від регістру** (для порівнянь, підрахунків, пошуку): 

```python
text = "Python"
len(text)         # 6
text.upper()      # "PYTHON"
text.lower()      # "python"
```

Техніка «перевести в нижній регістр і порівнювати» — стандарт для пошуку чи підрахунку символів, слів, тощо.

***

### Обрізання зайвих пробілів: strip

Користувачі часто вводять зайві пробіли перед і після тексту. `strip()` дає змогу їх прибрати: 

```python
raw = "   hello  "
clean = raw.strip()   # "hello"
```

Це особливо корисно при введенні логінів, email, назв файлів — там пробіли зазвичай не мають значення, але можуть ламати перевірки.

***

### Пошук і заміна: replace

`replace(old, new)` створює новий рядок, де всі входження `old` замінені на `new`:

```python
text = "Python is hard, but Python is fun"
text2 = text.replace("Python", "JavaScript")
# "JavaScript is hard, but JavaScript is fun"
```

Це найпростіший інструмент для «масового редагування» рядка: замінити певні слова, символи, маркери.

***

### split і join: текст ↔ список слів

Дуже часто нам потрібно: 

- розбити текст на частини (слова, поля, токени);  
- потім зібрати їх назад в інший формат.

#### split()

```python
text = "Python is fun and powerful"
words = text.split()   # ["Python", "is", "fun", "and", "powerful"]
```

За замовчуванням `split()` ділить по пробілах, але можна задавати і свій роздільник.

#### join()

```python
result = "-".join(words)  # "Python-is-fun-and-powerful"
```

`sep.join(iterable)` бере список рядків і склеює їх в один рядок, вставляючи `sep` між ними. 

Разом `split` і `join` — це «конвертер» між «рядок як текст» і «рядок як список слів».

## Обробка і аналіз тексту

### Перебір символів

Рядок — це послідовність, тому його можна перебирати в циклі:

```python
text = "hello"
for ch in text:
    print(ch)
```

Це основа для:

- підрахунку голосних чи певних символів;  
- побудови словника частот символів;  
- фільтрації символів (прибрати пробіли, спецсимволи тощо).

### Паліндроми

Паліндром — рядок, який читається однаково вперед і назад. Класичний прийом: 

1. Очистити рядок (прибрати пробіли, перевести до нижнього регістру).  
2. Порівняти з його розворотом:

```python
cleaned = text.replace(" ", "").lower()
is_palindrome = cleaned == cleaned[::-1]
```

Зріз `[::-1]` — короткий спосіб розвернути рядок.

## Простий синтаксичний аналіз: примітивна перевірка email

Реальна валідація email складна й зазвичай робиться регулярними виразами або готовими бібліотеками. 

У цьому модулі використовується **спрощена** версія — як вправа на роботу з рядками:

- перевірити наявність `"@"`;  
- знайти крапку `"."` **після** `"@"`.  

```python
email = input("Email: ")
at_index = email.find("@")

if at_index == -1:
    print("Invalid")
else:
    dot_index = email.find(".", at_index)
    if dot_index == -1:
        print("Invalid")
    else:
        print("Valid")
```

Це показує, як можна використовувати позиції символів і просту логіку над рядком, не заглиблюючись у regex.

## Текст як дані: частоти символів і слів

Поширений патерн у задачах обробки тексту: 

1. Є рядок.  
2. Ми хочемо дізнатися, як часто зустрічається кожен символ або кожне слово.  

Для символів:

```python
freq = {}
for ch in text:
    if ch in freq:
        freq[ch] += 1
    else:
        freq[ch] = 1
```

Для слів — те саме, але спочатку `text.split()`.

Це базовий крок до аналізу тексту: пошуку найчастіших слів, символів, побудови простих статистик.

## Навіщо цей модуль потрібен далі

Рядки проникають у всі наступні теми Python: 

- читання і парсинг файлів (логів, CSV, JSON);  
- обробка HTTP‑запитів і відповідей (URL, заголовки, JSON‑тіла);  
- побудова CLI‑інтерфейсів і повідомлень користувачеві;  
- простий текстовий препроцесинг перед аналізом даних або NLP.

У цьому модулі головне — навчитися дивитися на рядок **як на дані**, з якими можна:

- нормалізувати (регістри, пробіли);  
- перетворювати (split/join, replace);  
- перевіряти (email, паліндром, наявність символів);  
- аналізувати (підраховувати частоти, шукати закономірності).
