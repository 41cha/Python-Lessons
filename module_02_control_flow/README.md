# Module 02: Control Flow

У цьому модулі студенти вчаться керувати **потоком виконання** програми за допомогою умов: `if`, `elif`, `else`, а також логічних операторів `and`, `or`, `not`. Це дозволяє програмі приймати рішення залежно від даних, а не просто виконувати код зверху вниз.

## Навчальні цілі

Після проходження модуля студент зможе:

- Писати прості та складені умовні оператори `if`, `elif`, `else`.

- Обробляти декілька варіантів поведінки програми на основі різних умов (наприклад, система оцінювання, логін, вибір операції).
- Уникати типових помилок: неправильний порядок перевірок, пропущений `else`, зайві умови тощо.


## Теми модуля

У цьому модулі розглядаються такі теми керування потоком виконання:

- Базовий оператор `if`.

## Теорія: умовні оператори

### Оператор if

Базова форма умовного оператора:

```python
if condition:
    # блок коду, який виконається, якщо умова істинна
    ...
```

`condition` – це вираз, який повертає `True` або `False`. Якщо умова істинна, виконується блок під `if`, інакше програма переходить далі.

Приклад:

```python
age = 20
if age >= 18:
    print("Adult")
```

Якщо `age` більше або дорівнює 18, буде виведено `"Adult"`.

***

### if … else

Щоб обробити обидва варіанти (коли умова істинна і коли ні), використовують `if … else`:

```python
number = 5
if number % 2 == 0:
    print("Even")
else:
    print("Odd")
```

Тут `else` виконується тільки тоді, коли умова в `if` хибна.

***

### if … elif … else

Коли варіантів більше, ніж два, використовують ланцюжок `if / elif / else`:

```python
score = 75

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
elif score >= 60:
    print("D")
else:
    print("F")
```

У такому ланцюжку:

- Перевірки виконуються зверху вниз.
- Як тільки якась умова стає істинною, виконується відповідний блок, а решта умов уже не перевіряються.
- `else` завжди пишуть останнім і він виконується, якщо не спрацювала жодна попередня умова.

***

### Логічні оператори and, or, not

Логічні оператори дозволяють поєднувати кілька умов в один вираз:

- `and` – істинно, якщо **обидві** умови істинні.
- `or` – істинно, якщо істинна **хоча б одна** з умов.
- `not` – заперечує значення (`not True` → `False`, `not False` → `True`).

Приклади:

```python
age = 25
has_ticket = True

if age >= 18 and has_ticket:
    print("You can enter")

if age < 18 or not has_ticket:
    print("You cannot enter")
```

Логічні оператори часто використовують у задачах авторизації, перевірки діапазонів, фільтрації даних тощо.

## Огляд вправ модуля

У цьому модулі 6 практичних задач, які покривають різні типи умов та комбінації операторів.

### Task 1: Even or odd

- Ввід: одне ціле число.
- Вивід: `"Even"` для парних чисел, `"Odd"` для непарних.
- Цілі: використання `%` (остача від ділення), простий `if/else`.


### Task 2: Positive, negative or zero

- Ввід: одне число (може бути з комою).
- Вивід: `"Positive"`, `"Negative"` або `"Zero"`.
- Цілі: ланцюжок `if / elif / else`, правильний порядок умов.


### Task 3: Grading system

- Ввід: оцінка від 0 до 100.
- Вивід: літера `A`, `B`, `C`, `D` або `F` залежно від діапазону.
- Цілі: побудова «драбинки» умов, перевірка діапазонів зверху вниз.

Подібні приклади використовують у навчальних статтях про умовні оператори з системою оцінювання.

### Task 4: Simple login system

- Ввід: логін та пароль.
- Вивід: `"Login successful"` при правильних даних, `"Invalid credentials"` – при помилці.
- Цілі: логічний `and`, порівняння рядків, простий сценарій авторизації.


### Task 5: Max of three numbers

- Ввід: три числа.
- Вивід: найбільше з них.
- Цілі: порівняння кількох значень, покрокове оновлення змінної `max_value`, практика вкладених або послідовних `if`.


### Task 6 (optional): Simple calculator with operator

- Ввід: два числа та символ операції (`+`, `-`, `*`, `/`).
- Вивід: результат обчислення або повідомлення про помилку для невідомої операції.
- Додатково: обробка ділення на нуль.
- Цілі: розгалуження за значенням змінної (`if/elif/else`), робота із символами, обробка помилкових випадків.
