# Курс: Функції в Python (без коду)

Цей розділ знайомить з функціями — повторно використовуваними блоками коду. Учні навчаться створювати, викликати функції та передавати дані.

## 1. Що таке функції?

Функції — це "інструкції з назвою", які виконуються тільки при виклику. Використовуються для уникнення повторення коду.

**Переваги:**

- Код коротший і зрозуміліший
- Легко тестувати окремі частини
- Можна викликати багато разів з різними даними

**Структура:**

1. `def` + назва функції
2. Дужки з параметрами (або порожні)
3. Двоеточие і відступлений блок коду
4. `return` для повернення результату (необов'язково)

**Практика:** Створіть порожню функцію `привітання()`.

## 2. Прості функції без параметрів

Функції можуть виконувати дії без введених даних.

**Як створювати:**

1. Напишіть `def назва():`
2. Всередині — дії (print, обчислення)
3. Викликайте назвою: `назва()`

**Практика:** Створіть функцію, яка виводить ваше повне привітання.

## 3. Функції з параметрами

Параметри — дані, які передаються у функцію.

**Як використовувати:**

1. `def назва(параметр1, параметр2):`
2. Всередині використовуйте параметри як змінні
3. Виклик: `назва(значення1, значення2)`

**Практика:** Функція привітання з параметром імені.

## 4. Функція з return

`return` повертає результат для використання в інших місцях.

**Важливо:** Після `return` функція завершується!

**Практика:** Функція, яка повертає квадрат числа.

## Задача 1: Перевірка парності

**Мета:** Функція приймає число, повертає True (парне) або False.

### Покроковий план:

1. Створіть функцію `є_парне(число)` з одним параметром
2. Перевірте, чи число % 2 == 0
3. Поверніть True або False через return
4. Викличте функцію з кількома числами і виведіть результат
5. Протестуйте: 4 → True, 7 → False

**Підказка:** Використовуйте if/else всередині функції.

## Задача 2: Факторіал числа

**Мета:** Функція обчислює n! (1×2×3×...×n).

### Покроковий план:

1. Створіть функцію `факторіал(n)`
2. Ініціалізуйте результат = 1
3. Циклом від 1 до n помножуйте результат на i
4. Поверніть результат
5. Запросіть число у користувача, викличіть функцію
6. Протестуйте: 5! = 120, 0! = 1

**Підказка:** Для 0 повертайте 1 одразу.

## Задача 3: Гра "Вгадай число"

**Мета:** Функція загадує число, перевіряє вгадування з підказками.

### Покроковий план:

1. Створіть функцію `вгадай_число()` без параметрів
2. Всередині: import random, число = random.randint(1, 100)
3. Цикл while: запитайте вгадування у input()
4. Порівняйте: менше/більше/правильно (вихід з циклу)
5. Поверніть кількість спроб
6. Викличіть функцію і виведіть результат

**Підказка:** Використовуйте int(input()) для числа.

## Домашнє завдання

1. **Калькулятор як функція:** `обчислити(a, b, операція)` повертає результат
2. **Перевірка паліндрому:** `є_паліндром(рядок)` — читається однаково з обох боків
3. **Генератор паролів:** `згенерувати_пароль(довжина)` з цифр і букв
