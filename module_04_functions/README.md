## Теорія: функції в Python

### Створення функції

Функція створюється ключовим словом `def`:

```python
def greet(name):
    print(f"Hello, {name}!")
```

- `def` — оголошення функції.  
- `greet` — назва функції (повинна бути в `snake_case`).  
- `name` — параметр (змінна, яка отримує значення при виклику).  
- Відступлений блок — тіло функції.  

**Виклик функції:**

```python
greet("Andrii")  # виведе: Hello, Andrii!
```

***

### Функції з return

Функції можуть **повертати** значення через `return`:

```python
def circle_area(radius):
    return 3.1416 * radius * radius

area = circle_area(5)
print(area)  # 78.54
```

- `return` завершує виконання функції та повертає значення.  
- Якщо `return` не написано, функція повертає `None`.  
- Результат можна присвоювати змінним або передавати в інші функції.  

***

### Параметри функцій

#### Позиційні параметри
Параметри передаються в порядку оголошення:

```python
def info(name, age):
    print(f"{name} is {age} years old")

info("Andrii", 20)  # OK
# info(20, "Andrii")  # Помилка!
```

#### Параметри з значеннями за замовчуванням
Деякі параметри можуть мати значення за замовчуванням:

```python
def info(name, age=18, city="Unknown"):
    print(f"{name}, {age}, {city}")

info("Andrii")           # Andrii, 18, Unknown
info("Maria", 25)        # Maria, 25, Unknown
info("Oleh", 30, "Kyiv") # Oleh, 30, Kyiv
```

Параметри з `=` завжди пишуться **праворуч** від параметрів без значень за замовчуванням.

***

### *args — довільна кількість аргументів

Зірочка `*` перед параметром перетворює всі позиційні аргументи у **кортеж**:

```python
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3))      # 6
print(sum_all(10, 20, 30))   # 60
```

Усередині функції `numbers` — це кортеж `(1, 2, 3)`. Це дозволяє передавати будь-яку кількість чисел.

***

### Область видимості

**Локальні змінні** існують тільки всередині функції:

```python
def calculate():
    total = 0    # локальна змінна
    total += 10
    return total

result = calculate()
print(total)  # NameError! total не існує поза функцією
```

**Глобальні змінні** доступні всюди, але їх краще уникати:

```python
counter = 0  # глобальна

def increment():
    global counter
    counter += 1
```

Рекомендація: передавайте дані **через параметри**, а не через глобальні змінні.
