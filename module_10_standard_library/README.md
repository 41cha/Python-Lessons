# Module 10: Standard Library

Python постачається з великою **стандартною бібліотекою** — набором модулів, які закривають типові задачі: випадкові числа, дати, робота з файловою системою, формати даних (JSON) тощо.  У цьому модулі ви знайомитесь із кількома з найуживаніших: `random`, `datetime`, `os`/`os.path` та `json`.

## Навіщо стандартна бібліотека

Замість писати все «з нуля», ви підключаєте готові **добре протестовані** модулі:

- `random` — для генерації випадкових чисел і вибору випадкових елементів; 
- `datetime` — для роботи з датою і часом (сьогоднішня дата, форматування, різниця між датами); 
- `os` / `os.path` — для взаємодії з операційною системою: шляхи, список файлів, перевірка існування; 
- `json` — для збереження та завантаження структурованих даних у популярному текстовому форматі JSON. 

Це перший крок до «практичного» Python, де ваш код починає взаємодіяти з реальними файлами й даними, а не тільки з консоллю.

## random: випадковість у програмах

Модуль `random` дає можливість:

- генерувати випадкові числа (цілі й дійсні);  
- випадково обирати елемент зі списку.  

Базові приклади:

```python
import random

n = random.randint(1, 10)    # випадкове ціле 1..10
choice = random.choice(["rock", "paper", "scissors"])
```

Це використовується в:

- невеликих іграх (угадай число, камінь-ножиці-папір);  
- симуляціях, тестуванні, генерації даних.  

У вправах модуля ви якраз робите просту гру «відгадай число» і випадковий вибір зі списку.

## datetime: дати, час, відліки

`datetime` — один із ключових модулів Python: він дозволяє працювати з поточним часом, форматувати його і рахувати різницю між датами. 

Отримати поточні дату і час:

```python
import datetime

now = datetime.datetime.now()   # дата + час
today = datetime.date.today()   # тільки дата
```

Форматування:

```python
date_str = now.strftime("%Y-%m-%d")       # "2026-01-26"
time_str = now.strftime("%H:%M:%S")       # "22:10:05"
```

Різниця між датами:

```python
today = datetime.date.today()
new_year = datetime.date(today.year + 1, 1, 1)
delta = new_year - today
days_left = delta.days
```

У модулі ви використовуєте це, щоб:

- показати поточну дату/час у різних форматах;  
- порахувати, скільки днів залишилось до наступного Нового року;  
- додавати **timestamp** до лог-повідомлень.

## os і os.path: робота з файловою системою

Модуль `os` дає змогу «говорити» з операційною системою: 

- дізнатися поточну директорію:  

```python
import os
cwd = os.getcwd()
```

- подивитися, що в ній лежить:

```python
entries = os.listdir()   # список файлів і папок
```

Модуль `os.path` допомагає працювати з **шляхами до файлів**:

```python
import os

filename = "data.txt"
exists = os.path.exists(filename)
```

У вправах модуля ви:

- виводите поточну робочу директорію та список файлів у ній;  
- запитуєте ім’я файлу й перевіряєте, чи він існує.  

Це фундамент для програм, які працюють із реальними файлами: пошук, перевірка, обробка директорій.

## json: читати і зберігати структуровані дані

JSON — це текстовий формат, яким обмінюються майже всі веб‑сервіси. Він добре підходить для збереження словників і списків: `dict` ↔ JSON‑об’єкт, `list` ↔ JSON‑масив.

Модуль `json` вміє:

- **серіалізувати** Python-об’єкти в JSON‑рядок або файл (`json.dump`, `json.dumps`);  
- **десеріалізувати** JSON назад у Python-об’єкти (`json.load`, `json.loads`).  

Приклад запису в файл:

```python
import json

user = {"name": "Andrii", "age": 20}
with open("user.json", "w", encoding="utf-8") as f:
    json.dump(user, f, ensure_ascii=False, indent=2)
```

Приклад читання:

```python
with open("user.json", "r", encoding="utf-8") as f:
    data = json.load(f)
    # data — звичайний dict
```

У вправах ви створюєте словник з даними користувача, зберігаєте його в JSON-файл, а потім читаєте файл назад і друкуєте ключі/значення. Це готує до роботи з API, конфігами та простими базами даних на JSON.
