## Теорія модуля: базові поняття Python

### Що таке Python і де його застосовують

Python — це високорівнева мова програмування, яку активно використовують для веб‑розробки, автоматизації, аналізу даних, машинного навчання, створення скриптів та ігор.
Його люблять за простий, читабельний синтаксис, велику стандартну бібліотеку та величезну кількість готових сторонніх пакетів.

***

### Встановлення Python і середовище розробки

Щоб почати, потрібно:

- Встановити інтерпретатор Python з офіційного сайту python.org або через пакетний менеджер (наприклад, `apt`, `brew`, `choco`).
- Вибрати середовище розробки: часто використовують VS Code, PyCharm або Jupyter Notebook, бо вони підсвічують синтаксис і спрощують запуск коду.

Після встановлення в терміналі команда `python --version` (або `python3 --version`) має показувати номер версії.

***

### Запуск коду: REPL та скрипти

Є два базові способи запускати Python‑код:

- **Інтерактивний режим (REPL)**
Це коли відкривається консоль Python, і ви вводите команди рядок за рядком, одразу бачачи результат. Це зручно для експериментів та перевірки ідей.
- **Скрипт у файлі**
Код записується у файл, наприклад `main.py`, а потім запускається командою
`python main.py` (або `python3 main.py` залежно від системи).
Такий спосіб підходить для реальних програм, які зберігаються та виконуються багато разів.

***

### Змінні та присвоєння

**Змінна** — це ім’я, яке посилається на деяке значення в пам’яті (можна думати як про коробку з етикеткою).
Щоб створити змінну, достатньо записати:

```python
age = 20
name = "Andrii"
height = 1.75
```

- Ліва частина — назва змінної.
- Оператор `=` — оператор **присвоєння**, він записує значення праворуч у змінну ліворуч.

Правила іменування змінних у Python:

- можна використовувати літери, цифри та нижнє підкреслення `_`;
- ім’я не може починатися з цифри;
- не варто використовувати пробіли та спеціальні символи;
- рекомендовано використовувати англійські слова у форматі `snake_case` (наприклад, `user_name`, `total_price`).

***

### Базові типи даних

У цьому модулі розглядаються чотири базові типи:

- `int` — цілі числа, наприклад `5`, `0`, `-42`.
- `float` — числа з комою (дійсні), наприклад `3.14`, `0.5`, `-1.0`.
- `str` — рядки, тобто текст у лапках: `"Hello"`, `'Python'`.
- `bool` — логічні значення: `True` або `False`.

Щоб дізнатися тип змінної, використовують функцію `type()`:

```python
x = 10
print(type(x))  # <class 'int'>
```

Це корисно для розуміння, з якими даними працює програма.

***

### Ввід та вивід даних

Для виводу інформації на екран використовують функцію `print()`:

```python
print("Hello, world!")
print("Age:", 20)
```

`print()` може приймати кілька аргументів, автоматично розділяючи їх пробілом.

Для зчитування даних з клавіатури використовують `input()`:

```python
name = input("Введіть ім'я: ")
```

Функція `input()` завжди повертає рядок, тому числа потрібно перетворювати:

```python
age = int(input("Введіть вік: "))
height = float(input("Введіть зріст у метрах: "))
```

Таким чином, можна будувати діалогові програми, які працюють з даними користувача.

***

### Арифметичні оператори

Python підтримує стандартні математичні операції над числами:

- `+` — додавання: `3 + 2` → `5`
- `-` — віднімання: `5 - 1` → `4`
- `*` — множення: `4 * 2` → `8`
- `/` — ділення (результат завжди `float`): `5 / 2` → `2.5`
- `//` — цілочисельне ділення: `5 // 2` → `2`
- `%` — остача від ділення (modulo): `5 % 2` → `1`
- `**` — піднесення до степеня: `2 ** 3` → `8`

Порядок виконання операцій такий самий, як у математиці: спочатку степені, потім множення/ділення, потім додавання/віднімання. Для зміни порядку використовують дужки:

```python
result = (2 + 3) * 4  # 20
```

Ці оператори дозволяють будувати формули для конвертера температур, калькулятора, обчислення площі фігур тощо вже в першому модулі.
