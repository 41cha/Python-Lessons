# Module 01: Introduction and Basics

Цей модуль дає практичний **вступ** до Python для повних початківців. Студенти встановлять Python, запустять свою першу програму та навчаться працювати зі змінними, базовими типами даних і введенням з клавіатури, як у більшості базових курсів Python.[^1][^2]

## Навчальні цілі

Після проходження модуля студент зможе:

- Встановити Python та запустити `.py` файл з редактора або терміналу.[^3]
- Використовувати змінні для зберігання даних і розуміти базові типи: цілі числа, числа з комою, рядки, логічні значення.[^4]
- Використовувати `input()` для зчитування даних з клавіатури і `print()` для виведення результатів.[^5][^6]
- Виконувати базові арифметичні операції та форматувати вивід за допомогою f-рядків.[^6][^7]


## Теми модуля

У цьому модулі студент знайомиться з такими ключовими темами:

- Що таке Python і де його застосовують (автоматизація, веб, аналіз даних, ML, ігри тощо).[^8][^3]
- Встановлення Python і вибір середовища розробки (VS Code, PyCharm тощо).[^9]
- Запуск коду:
    - Інтерактивний режим Python (REPL).
    - Скрипти у файлі: команда `python main.py`.[^9]
- Змінні та присвоєння:
    - Правила іменування: літери, цифри, підкреслення, без початку з цифри.
    - Оператор присвоєння `=` і уявлення “змінна = коробка з етикеткою”.[^4]
- Базові типи даних:
    - `int`, `float`, `str`, `bool`.
    - Функція `type()` для перевірки типу змінної.[^7][^4]
- Ввід та вивід:
    - `print()` з кількома аргументами.
    - `input()` та перетворення через `int()` / `float()`.[^5][^6]
- Арифметичні оператори:
    - `+`, `-`, `*`, `/`, `//`, `%`, `**`.
    - Порядок виконання операцій та дужки.[^7]


## Структура файлів модуля

У папці `module_01_introduction_and_basics/` використовуються такі файли:

- `README.md` – теорія, пояснення та інструкції до модуля.[^10]
- `exercises.py` – умови задач з короткими підказками, без готових рішень.[^11]
- `solutions.py` – повні розв’язки для викладача або самоперевірки після спроби виконання задач.[^12]

Така структура допомагає студентам спочатку самостійно розв’язувати задачі, а вже потім звірятись з готовим кодом.

## Огляд вправ

У модулі є 5 базових вправ, спеціально підібраних для практики змінних, введення, виведення та простих виразів. Подібний формат застосовують у багатьох вступних практикумах.[^13][^11]

- **Task 1: Simple calculator**
    - Запитати у користувача два числа.
    - Вивести їхню суму, різницю, добуток і частку.
    - Цілі: `input()`, `float()`, арифметика, f-рядки.
- **Task 2: Celsius to Fahrenheit converter**
    - Зчитати температуру в градусах Цельсія та перетворити її у Фаренгейти за формулою $F = C \times 9/5 + 32$.
    - Цілі: числові вирази, форматований вивід з одним знаком після коми.[^14][^15]
- **Task 3: Personal info**
    - Запросити ім’я, вік та зріст.
    - Вивести відформатоване привітання одним рядком.
    - Цілі: рядки, перетворення типів (`int`, `float`), f-рядки.[^1][^6]
- **Task 4: Rectangle area and perimeter**
    - Зчитати довжину та ширину прямокутника.
    - Обчислити площу і периметр, вивести їх з точністю до двох знаків.
    - Цілі: використання формул, арифметичні вирази, форматування.[^13][^7]
- **Task 5: Data types**
    - Створити чотири змінні різних типів і вивести їх типи за допомогою `type()`.
    - Цілі: розуміння `int`, `float`, `str`, `bool` і перевірка типів.[^4]


## Як працювати з модулем

Рекомендований порядок роботи для студента:

1. Відкрити `exercises.py` та уважно прочитати умову й підказки до кожної задачі.
2. Реалізувати розв’язок безпосередньо під коментарем `# TODO: your code here`.
3. Запускати `exercises.py` кілька разів з різними вхідними даними та перевіряти поведінку програми.
4. Лише після реальних спроб відкрити `solutions.py` і порівняти власний код з запропонованим рішенням.
5. Спробувати покращити свій код, орієнтуючись на приклади зі `solutions.py` (імена змінних, форматування, обробка крайніх випадків).[^10][^12]
