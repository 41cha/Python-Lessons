## Коротко про основні структури

### Список (list)

Список — це впорядкована змінна колекція елементів у квадратних дужках `[]`. 

```python
numbers = [10, 20, 30]
numbers.append(40)
numbers [dataquest](https://www.dataquest.io/blog/data-structures-in-python/) = 25
```

- Дозволяє дублікати.  
- Елементи можна змінювати, додавати, видаляти.  
- Доступ за індексом: `numbers[0]`.  

### Кортеж (tuple)

Кортеж — це впорядкована **незмінна** колекція в круглих дужках `()`.

```python
point = (3, 5)
x, y = point  # розпакування
```

- Зручно для фіксованих даних (наприклад, координати).  
- Не можна змінити після створення (додає безпечності, трохи швидший за список).  

### Множина (set)

Множина — це неупорядкована колекція **унікальних** елементів у фігурних дужках `{}`.

```python
nums = {1, 2, 2, 3}
# фактично: {1, 2, 3}
```

- Автоматично прибирає дублікати.  
- Застосовується для перевірки належності, унікальних значень, операцій перетину/об’єднання. 

### Словник (dict)

Словник — це колекція пар `ключ: значення` у фігурних дужках `{}`.

```python
student = {"name": "Andrii", "age": 20}
print(student["name"])
student["age"] = 21
```

- Швидкий доступ до значення за ключем.  
- Дуже корисний для зберігання структурованих даних та підрахунків (наприклад, частоти слів).

## Огляд вправ модуля

### Task 1: Basic list operations

- Створення списку чисел, додавання нового елемента, видалення елемента.  
- Вивід довжини, першого та останнього елементів.  
- Тренуються: методи `.append()`, `.pop()`, індекси, `len()`.

### Task 2: List slicing

- Створення списку слів і робота зі зрізами: перші 3, останні 2, список у зворотному порядку.  
- Тренуються: `words[:3]`, `words[-2:]`, `words[::-1]`.  

### Task 3: Remove duplicates using set

- Є список з повторами, потрібно прибрати дублікати через множину і повернути результат у вигляді списку.  
- Тренуються: перетворення `list -> set -> list`, ідея «унікальні елементи».

### Task 4: Tuple for coordinates

- Створення кортежу `point = (x, y)`, доступ за індексом і розпакування у змінні `x, y`.  
- Тренуються: робота з кортежами, розпакування (`x, y = point`).

### Task 5: Dictionary of student

- Створення словника `student` з полями `name`, `age`, `city`.  
- Вивід значень, оновлення міста та додавання нового поля `grade`.  
- Тренуються: доступ до значень `dict[key]`, додавання та зміна пар ключ–значення.

### Task 6: Word frequency (dict)

- Рядок `"python is fun and python is easy"` потрібно розбити на слова та порахувати частоту кожного слова.  
- Результат — словник `{слово: кількість}`.  
- Тренуються: `split()`, цикл по словах, патерн «словник лічильників»: якщо слово вже є — збільшити, інакше створити з 1.

### Task 7: Set operations

- Є множини `a` та `b`, треба знайти:  
  - перетин `a & b`;  
  - об’єднання `a | b`;  
  - різницю `a - b`.  
- Тренуються: базові операції над множинами для пошуку спільних, усіх та унікальних елементів.

### Task 8 (optional): List of dictionaries

- Список `students`, де кожен елемент — словник `{"name": ..., "age": ...}`.  
- Потрібно вивести всі імена та знайти середній вік.  
- Тренуються: комбінація списків і словників, ітерація по колекції складних об’єктів, підрахунок середнього.
